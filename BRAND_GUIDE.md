## 2. Color Palette

Our color scheme reflects professionalism (Navy) and energy/warmth (Gold).

### Primary Colors

| Color Name | Hex Code | Purpose |
| :--- | :--- | :--- |
| **JPD Navy** | `#003B5C` | **Primary Brand Color.** Used for headers, primary buttons, and main brand elements. |
| **JPD Gold** | `#F5D042` | **Accent Color.** Used for Call-to-Actions (CTAs), highlights, underlines, and attention-grabbing elements. |

### Secondary & Neutral Colors

These colors support the primary palette to create a clean, readable interface.

| Color Name | Hex Code | Purpose |
| :--- | :--- | :--- |
| **Deep Navy** | `#012C45` | **Interactive State.** Used for hover states on primary buttons or dark footers. |
| **Slate** | `#4A4F57` | **Body Text.** Softer than pure black for better readability on screens. |
| **Off-White** | `#F8F9FA` | **Backgrounds.** Used as the main page background to reduce eye strain compared to pure white. |
| **Pure White** | `#FFFFFF` | **Cards & Containers.** Used for content areas to make them pop against the off-white background. |

## 3. Typography

We use a modern, geometric type scale that balances readability with strong personality.

### Headings: **Outfit**
Used for all hierarchical headings (H1 - H6). It is a geometric sans-serif that appears modern and friendly yet technical.

*   **Weight:** Bold (700) or Semi-Bold (600)
*   **Usage:** Page titles, section headers, card titles.
*   **Tracking:** Slightly increased (`0.02em`) for clarity.

### Body Copy: **Inter**
Used for all paragraph text, input fields, and UI elements. It is optimized for computer screens and high legibility.

*   **Weight:** Regular (400) or Medium (500)
*   **Usage:** Paragraphs, lists, descriptions, navigation links.

## 4. Design Elements & Usage

### Buttons

*   **Primary Button (Action):**
    *   **Background:** Navy (`#003B5C`)
    *   **Text:** White
    *   **Hover:** Deep Navy (`#012C45`)
    *   *Usage: "Contact Us", "Submit", "Read More"*

*   **Highlight Button (High Priority):**
    *   **Background:** Gold (`#F5D042`)
    *   **Text:** Navy (`#003B5C`)
    *   **Hover:** Lighter Yellow/Gold
    *   *Usage: "Book Now", "Emergency Call", "Get a Quote"*

*   **Outline Button (Secondary):**
    *   **Border:** Navy (`2px solid #003B5C`)
    *   **Text:** Navy
    *   **Hover:** Fill Navy, Text White
    *   *Usage: "View Services", "Learn More"*

### Section Headers
Section headers often feature a **Gold Underline** design element to separate them from content and draw the eye.

*   **Style:** Relative positioning with a pseudo-element underline.
*   **Dimensions:** `w-20` (approx 80px) width, `h-1` (4px) height.
*   **Color:** Gold (`#F5D042`).

### Cards (Services/Reviews)
*   **Background:** White (`#FFFFFF`)
*   **Shadow:** Subtle shadow (`shadow-md`) increasing on hover (`shadow-xl`).
*   **Border:** Thin grey border (`border-gray-100`) for definition.
*   **Rounded Corners:** `rounded-xl` for a modern, approachable feel.

---

## 5. Logo Usage
*   **Light Backgrounds:** Use the full colored logo (Navy text).
*   **Dark Backgrounds:** Use the reversed logo (White text) if available, or ensure sufficient contrast.
*   **Icon:** The "JPD" mark or lightning bolt motif can be used as a standalone favicon or avatar.
